What Filament Does

    
Onchain identity,
stitched together

    
A single person may control dozens of wallets across chains. Filament finds them using pure behavioral signals — no personal data required.

  

  

    
Identity Stitching
Links wallets across 7 chains using 9 independent behavioral heuristics, aggregated into a single confidence score.

    
Bridge Hop Tracing
Follows capital movement across Stargate, Hop, Across, and LayerZero at configurable graph depth.

    
Temporal Correlation
Detects wallets active at identical unusual hours — sleep cycles and burst patterns are unique per operator.

    
Deployment DNA
Extracts bytecode fingerprints and CREATE2 salt patterns to detect wallets deploying identical contracts.

    
Sanctions Exposure
Maps cluster exposure to sanctioned addresses at configurable depth — 2nd and 3rd degree included.

    
Gas Fingerprinting
Gas bidding behavior — tip precision, base fee preference — is surprisingly unique per operator.

  










  

    
How It Works

    
From wallet to
identity cluster

  

  

    
01 —
Submit a Wallet
Pass any EVM wallet address and select which chains and graph depth to investigate.

    
02 —
Heuristics Fan Out
9 behavioral modules run in parallel — nonce patterns, gas signatures, bridge hops, deployment DNA, and more.

    
03 —
Scores Aggregate
The confidence engine combines weighted scores into a single tier — High, Probable, or Speculative.

    
04 —
Cluster Returned
Every linked wallet is returned with full evidence — every signal that fired, every score that contributed.

  










  

    
MCP Tools

    
11 tools.
One investigation layer.

    
Each tool is independently callable via the Model Context Protocol. AI agents can compose them into any workflow.

  

  

    
stitch_identity
Primary identity stitching across chains with full confidence report.

    
nonce_pattern_match
Temporal fingerprint similarity from nonce sequencing and timing gaps.

    
deployment_dna_scan
Bytecode fingerprints, CREATE2 salts, and init code hash matching.

    
bridge_hop_tracer
Directed fund flow graph across Stargate, Across, Hop, LayerZero.

    
gas_behavior_fingerprint
Gas bidding style classification and cross-chain similarity score.

    
eoa_cluster_graph
Funding adjacency graph with edge weights and central wallet detection.

    
contract_interaction_overlap
Obscure contract interaction overlap scoring between two wallets.

    
temporal_activity_correlation
Cross-correlation matrix of activity timestamps across wallets.

    
entropy_address_scorer
Low-entropy derivation pattern detection in child addresses.

    
sanction_proximity_mapper
Exposure tree mapping to sanctioned addresses at N hops.

    
identity_confidence_report
Full aggregated report with tier, breakdown, evidence, and graph payload.
